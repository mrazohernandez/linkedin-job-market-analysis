# 📊 LinkedIn Job Market Analysis (Data Roles)

## 📌 Overview
This project analyzes 100,000 LinkedIn job postings to identify trends in data-related roles, including in-demand skills, top hiring companies, and key locations.

The analysis focuses specifically on data roles such as Data Analyst, Data Scientist, and Data Engineer by filtering job titles using SQL.

---

## 🛠️ Tools Used
- SQL (PostgreSQL)
- Tableau
- Excel

---

## 📈 Key Insights
- Excel remains the most in-demand skill across data roles  
- SQL and Python are core technical requirements  
- Hiring demand is concentrated in major metro areas such as New York, Chicago, and Houston
